# MLP Hidden Layer Analysis

## 📌 Overview
This project explores how the number of hidden layers in a neural network affects performance.

## 🎯 Objective
To compare neural networks with different depths and analyze their accuracy.

## 📊 Dataset
- Digits dataset from Scikit-learn
- 8x8 grayscale images of digits (0–9)

## ⚙️ Models Used
- 1 Hidden Layer
- 2 Hidden Layers
- 3 Hidden Layers

## 📈 Results
The experiment shows that increasing depth improves performance up to a certain point, after which gains become limited or decrease.

## 🚀 How to Run

1. Install dependencies: pip install -r requirements.txt

2. Run the notebook: jupyter notebook notebook.ipynb

## 📁 Files
- `notebook.ipynb` → Code implementation
- `tutorial.pdf` → Full report
